# Квантрон
## Определение
Квантрон — это элементарная ячейка дискретного пространства (квант), которая содержит внутри себя микроконтинуум с одной из трёх возможных топологий: 3D-шар (S³), связка 2D-сферы/тора с 1D-кольцом (S²/T² + S¹), или три взаимно перпендикулярных 1D-кольца (3×S¹).
## Пояснение
Квантрон не является "частицей" в классическом смысле. Это структурная единица пространства, заполненная вложенной вселенной (микроконтинуумом). Разные топологии заполнения дают разные типы квантронов (Q₀, Q₁, Q₂, Q₃), которые служат строительными блоками для всех наблюдаемых частиц и полей.
## Примеры
- Q₀-квантрон: 3D-шар (тёмная материя)
- Q₁-квантрон: 1D-кольцо + 2D-сфера/тор (протокварки)
- Q₂-квантрон: смещённые 1D-кольцо и 2D-тор (латчеры)
## Связи
- Является основой для всех типов частиц в ВПК-модели.
- Используется в определениях: Qf-латчер (OBJ-011), Qν-латчер (OBJ-012), протокварк (OBJ-014).
