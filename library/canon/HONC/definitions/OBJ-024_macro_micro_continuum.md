# Макроконтинуум и Микроконтинуум
## Определение
Два соседних уровня иерархии вложенных континуумов. Микроконтинуум — вселенная, содержащаяся внутри кванта пространства (квантрона) текущего уровня. Макроконтинуум — вселенная более высокого уровня, для которой текущий континуум является квантом.
## Пояснение
Иерархия масштабов задаёт коэффициент масштабирования λ/τ₀ = c, где c — граничная скорость, общая для всех уровней.
