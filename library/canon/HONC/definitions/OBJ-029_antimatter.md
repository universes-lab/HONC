# Антиматерия (как инверсия системы координат)
## Определение
Состояние частицы, при котором её внутренняя система координат отражена (YXZ✇ вместо XYZ✇). Это меняет направление действия компоненты силы Лоренца, что для внешнего наблюдателя выглядит как противоположный электрический заряд. Аннигиляция — это топологический переход между зеркальными конфигурациями, а не уничтожение вещества.
## Пояснение
Отсутствует отдельный «антимир» — все частицы существуют в одном континууме, различаясь только ориентацией осей.
