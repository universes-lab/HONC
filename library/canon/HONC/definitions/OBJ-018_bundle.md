# Жгут (латчерная цепочка)
## Определение
Протяжённая структура, образованная последовательным соединением латчеров (Qf или Qν) через общее поле времени. Жгут является основным структурным элементом для фотонов, нейтрино и лептонов. Его топология (число осей, синхронизация вращений) определяет свойства соответствующей частицы.
## Пояснение
Фотон — жгут Qf-латчеров; нейтрино — жгут Qν-латчеров; лептон — жгут Qν + обвесы Qf.
