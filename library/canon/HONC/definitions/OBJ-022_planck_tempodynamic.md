# Темподинамическая постоянная Планка (ℏ_TD)
## Определение
Эффективная постоянная, связывающая дискретную структуру пространства-времени с квантовыми эффектами в темпоральной динамике. Определяется как ℏ_TD = I·λ·c, где I — инерциальность, λ — квант пространства, c — граничная скорость. В классическом пределе при корректной калибровке должна сводиться к постоянной Планка ℏ.
## Пояснение
**Связан с открытым вопросом AX8:** Прямая подстановка I (безразмерное число квантронов N) даёт размерность м²/с, а не действия. Требуется математический мост между dt/dx-формализом и классической размерностью [ML²T⁻¹].
