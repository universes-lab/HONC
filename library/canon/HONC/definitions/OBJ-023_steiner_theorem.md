# Теорема Штейнера (для латчеров)
## Определение
Применение классической теоремы о параллельных осях к смещённым центрам 1D-кольца и 2D-тора в латчере. Добавочный член Md² (где d — расстояние между осями) приводит к трём уровням суммарного гироскопического момента, соответствующим трём поколениям лептонов.
## Пояснение
Ключевой математический инструмент для объяснения иерархии масс лептонов без введения дополнительных параметров.
