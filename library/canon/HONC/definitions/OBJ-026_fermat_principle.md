# Принцип Ферма (обобщённый)
## Определение
Закон, согласно которому любое возмущение (включая фотоны и другие частицы) распространяется по пути, для прохождения которого требуется минимальное время. В темпоральной динамике это эквивалентно минимизации интеграла от поля времени вдоль траектории.
## Пояснение
Является основой для вывода гравитации как неоднородности поля времени и объясняет искривление траекторий без введения сил.
