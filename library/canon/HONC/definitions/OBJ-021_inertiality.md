# Инерциальность (I)
## Определение
Характеристика сопротивления топологической конфигурации изменениям градиента поля времени. Определяется через момент инерции и распределение поля времени внутри квантрона или жгута. В dt/dx-формализме I является первичной величиной, из которой выводится инертная масса.
## Пояснение
**Связь с открытым вопросом AX8:** Для перехода к классическому пределу требуется калибровка ℏ_eff = I·λ·c. Однако при I, выраженном в безразмерных единицах (число квантронов N), возникает размерностное противоречие с универсальностью ℏ. Требуется отдельная математическая разработка (Math Track).
