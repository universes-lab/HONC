# Граничная скорость (c)
## Определение
Постоянная, численно равная коэффициенту масштабирования между соседними уровнями иерархии континуумов: c = λ / τ₀. Является максимальной скоростью распространения возмущений в данном континууме. В классическом пределе соответствует скорости света.
## Пояснение
Не является фундаментальной константой в онтологическом смысле — она вытекает из геометрии дискретной решётки и иерархии.
